Adds an admin page where you can change oEmbed cache related settings and clear cached embed content.

== Description ==

Sometimes when you add emebed links to your posts WordPress fails while fetching embed content through oEmbed. This plugin gives you control over emebed settings and cache by letting you clear failed embeds, change cache life time and the time limit WordPress uses when fetching the embed code. 

This plugin requires PHP version >= 5.3.0

Plugin developed by 61 Degrees North: http://www.61degreesnorth.se

== Installation ==

1. Upload `plugin-name.php` to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can now access the Oembed Cache page from the settings menu in admin
